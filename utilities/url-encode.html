How to use

  • URL component: Encode query params and path segments (encodes / ? & = etc.)
  • Full URL: Encode the whole URL but preserve URL structure characters
  • Parse URL params: Extract query parameters from a URL
  • Supports non-ASCII and special characters

Note: All operations run locally in your browser. Nothing is uploaded.
